# ⚡ Skill Estrategia de Contenido

Skill de Claude para generar estrategias de contenido profesionales, accionables y listas para ejecutar.

Diseñada para creadores, marcas, coaches, agencias y negocios que quieren dejar de improvisar su contenido.

---

## ⚡ ¿Qué hace esta skill?

Convierte ideas sueltas en una estrategia de contenido completa:

- Ideas de contenido
- Hooks optimizados
- Pilares de contenido
- Formatos por red social
- Calendario listo para ejecutar
- CTAs estratégicos

Todo estructurado y listo para ejecutar en minutos.

---

## ⚡ Instalación (1 minuto)

1. Abre Claude
2. Ve a **Personalizar**
3. Selecciona **Crear Skill**
4. Copia y pega el contenido del archivo `estrategia-contenido.md` en los campos correspondientes:
   - Nombre
   - Descripción
   - Instrucciones
5. Guarda la skill
6. Pidele una estrategias de contenido y comenzará automáticamente
